**Insert Lists Anywhere in your site**

Basically Lists can be inserted anywhere in your site:

* Page/post WYSIWYG editor by using the buttion in the editor
* In sidebars by using the OTW Shortcode Widget
* In template files by using the List's shortcode and WordPress do_shortcode function

**Custom styling**

If you need to further style a List here are your options:

* Each List shortcode has it's unique CSS class that can be used to style all Lists.
* Create a new class for each instance of a List shortcode in its interface so you can style it individually.
